# GankioKotlin
练手项目， 使用Kotlin语言开发的一个干货集中营Demo 

# 项目简介
1、首页使用PagerSlidingTabStrip + ViewPager实现页面左右滑动

2、网络请求采用Retrofit ，自定义CallBack 对请求返回数据统一处理

3、图片加载使用Glide 
...
